### Quick Principles

- **Writing experience first**: Ultra-low latency, distraction-free focus, excellent typography, flexible components.
- **Fast search**: Comprehensive, sub-100ms search across all content types.
- **Privacy first**: Transparent, predictable data practices with user control. Local-first by default.
- **Local-first**: All data local by default. Optional user-chosen backup.
- **Sustainable**: 7-day free trial, $45 lifetime.
