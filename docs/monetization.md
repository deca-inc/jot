## Monetization

### Trial

- 7-day free trial starting at first app launch. Local clock + monotonic time with tamper checks.

### Purchase

- Subscription based and/or one $X lifetime license. Local license file + optional accountless receipt.
- Device policy: allow multiple devices for the same user (2-3), offline friendly.

### UX

- Discreet banner post-trial with graceful limits (no hard lock on reading/export).
- Purchase flow integrated in Settings and timeline banner.
